### Waylandを￹使用￺しよう￻するばあい

　Ubuntu 21.04￹以降￺いこう￻や、Fedora 25￹以降￺いこう￻では、￹画面￺がめん￻の￹描画￺びょうが￻にデフォルトではWaylandをつかうようになっています。

　IBusをWaylandで使用するばあいは、￹環境￺かんきょう￻￹変数￺へんすう￻GTK_IM_MODULEにibusを￹指定￺してい￻してください。
