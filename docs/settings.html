設定(せってい)のかえかた

「ひらがなIMEの設定(せってい)」ウィンドウ

　「ひらがなIME」の設定(せってい)は、「ひらがなIMEの設定(せってい)」ウィンドウでおこないます。トップバーの現在(げんざい)の入力(にゅうりょく)モードを表示(ひょうじ)している部分(ぶぶん)（「あ▼」、「A ▼」など）をクリックすると、入力(にゅうりょく)ソースのメニューがひらきます。（キーボード配列(はいれつ)や入力(にゅうりょく)メソッドのことをまとめて「入力(にゅうりょく)ソース」とよんでいます。） そこから[設定(せってい)]をえらぶと、「ひらがなIMEの設定(せってい)」ウィンドウがひらきます。


　「ひらがなIMEの設定(せってい)」ウィンドウには、つぎのふたつのタブがあります。

タブのなまえ 説明(せつめい)
キーボード 日本語(にほんご)の入力(にゅうりょく)方法(ほうほう)や使用(しよう)するキーボードの種類(しゅるい)をえらびます。
辞書(じしょ) 使用(しよう)する漢字(かんじ)辞書(じしょ)をえらびます。「ひらがなIME」は、学年(がくねん)別(べつ)の漢字(かんじ)辞書(じしょ)を用意(ようい)しています。

　設定(せってい)した内容(ないよう)は、[適用(てきよう)]ボタンをおすと、「ひらがなIME」にすぐ反映(はんえい)されます。

[キーボード] タブ

　[キーボード]タブでは、日本語(にほんご)の入力(にゅうりょく)方法(ほうほう)を設定(せってい)します。


　「打(う)ちかた」は、つぎの３種類(しゅるい)からえらびます。

打(う)ちかた 説明(せつめい)
ローマ字(じ) ローマ字(じ)で入力(にゅうりょく)します。
かな(JIS配列(はいれつ)) JIS配列(はいれつ)をつかって、かなで入力(にゅうりょく)します。
かな(ニュースティックニー配列(はいれつ)) ニュースティックニー配列(はいれつ)をつかって、かなで入力(にゅうりょく)します。

　キーボードの「種類(しゅるい)」には、つかっているキーボードの種類(しゅるい)を指定(してい)します。「ひらがなIME」は、「日本語(にほんご)キーボード」と「英語(えいご)(US)キーボード」に対応(たいおう)しています。「打(う)ちかた」を「かな(JIS配列(はいれつ))」にしたときは、いずれにしても「日本語(にほんご)キーボード」になります。

パソコンにつなげるキーボードの種類(しゅるい)を変更(へんこう)したときは？

　パソコンにつなげるキーボードの種類(しゅるい)を変更(へんこう)したときは、入力(にゅうりょく)ソースの再(さい)設定(せってい)が必要(ひつよう)です。まず、キーボードにあわせて「英語(えいご)(US)」か「日本語(にほんご)」かどちらかを選択(せんたく)してください。そのあとで、「日本語(にほんご)(Hiragana IME)」を選択(せんたく)しなおします。「ひらがなIME」は、直前(ちょくぜん)に選択(せんたく)していた入力(にゅうりょく)ソースのキーボード配列(はいれつ)をそのままひきつぎます。「ひらがなIMEの設定(せってい)」の「種類(しゅるい)」をきりかえても、英語(えいご)キーボードと日本語(にほんご)キーボードの変更(へんこう)ができるわけではないので注意(ちゅうい)してください。

ローマ字(じ)の「ん」のつづりかた

　「ん」をいつも「nn」で入力(にゅうりょく)したいときは、『'nn'をかならず「ん」に変換(へんかん)する』をオンにします。

　ワープロでは、「nn」と入力(にゅうりょく)すると「ん」になる便法(べんぽう)がつかわれてきました。ふたつめの「n」は、日本語(にほんご)キーボードではおしにくい❲'❳のかわりです。現在(げんざい)は、このワープロの方式(ほうしき)をさだめたJIS規格(きかく)は廃止(はいし)されています。


参考(さんこう): 『JIS X 4063の廃止(はいし)』。

[辞書(じしょ)] タブ

　[辞書(じしょ)]タブでは、漢字(かんじ)変換(へんかん)につかう辞書(じしょ)を指定(してい)します。「ひらがなIME」は、おとな用(よう)の漢字(かんじ)辞書(じしょ)のほかに、小中学生(しょうちゅうがくせい)用(よう)の学年(がくねん)別(べつ)の辞書(じしょ)を用意(ようい)しています。


　漢字(かんじ)辞書(じしょ)は、「漢字(かんじ)辞書(じしょ)」コンボボックスでえらびます。

　おとな用(よう)の辞書(じしょ)は、「常用(じょうよう)漢字(かんじ)表(ひょう)」にある漢字(かんじ)と「人名(じんめい)用(よう)漢字(かんじ)表(ひょう)」にある漢字(かんじ)をつかって構成(こうせい)されています。学年(がくねん)別(べつ)の辞書(じしょ)は、平成(へいせい)29年(ねん)の「音訓(おんくん)の小(しょう)・中(ちゅう)・高等学校(こうとうがっこう)段階(だんかい)別(べつ)割(わ)り振(ふ)り表(ひょう)」にそって構成(こうせい)されています。

　おくりがなのつけかたは、『送(おく)り仮名(かな)の付(つ)け方(かた)』に本則(ほんそく)や許容(きょよう)されているつけかたがしめされています。「おとな用(よう)(おくりがな: 本則(ほんそく))」辞書(じしょ)では、本則(ほんそく)をつかいます。「おとな用(よう)(おくりがな: 本則(ほんそく)+許容(きょよう))」辞書(じしょ)では、本則(ほんそく)にくわえて、許容(きょよう)されているおくりかたもつかえます。


例(れい)) 「おとな用(よう)(おくりがな: 本則(ほんそく)+許容(きょよう))」辞書(じしょ)をつかうと、「落(おと)す」のような変換(へんかん)もできます。

お―とす❲変換❳ → 落とす   # 原則
おと―す❲変換❳ → 落す     # 許容

個人(こじん)用(よう)辞書(じしょ)

　「ひらがなIME」の標準(ひょうじゅん)の漢字(かんじ)辞書(じしょ)にない語(ご)は、個人(こじん)用(よう)の辞書(じしょ)ファイルに登録(とうろく)できます。個人(こじん)用(よう)の辞書(じしょ)ファイルは、ディレクトリ ~/.local/share/ibus-hiragana/ のなかに保存(ほぞん)されています。はじめは my.dic というなまえになっています。
